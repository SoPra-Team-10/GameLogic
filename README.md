## Installing
First you need to install [SopraMessage](https://github.com/SoPra-Team-10/Messages).

In the root directory of the project run
```
sudo make install
```
the library can now be included using

```
#include <SopraGameLogic/XY>
```

and linked using

```
-lSopraGameLogic
```
